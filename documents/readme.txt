To test the system, download or clone the project-

Have Node.js, MongoDB installed in your system

Open the cmd and nevigate to g-help Project folder, from here do npm start, this starts your Node.js server and listen at 3339

Open cmd and cd to bin directory of mongoDB. type mongod --dbpath path to your g-help\db and hit enter.

Open another cmd and cd to bin directory of mongoDB. type mongo hit enter and then type use ghelp. your db is now data directory of g-help

Go to localhost:3339 in your browser to test
